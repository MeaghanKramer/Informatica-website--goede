What is Machine Learning?

Form of artificial intelligence (Al) that mainly focuses on learning from experience and making predictions. Machine Learning is a form of AI and is based on the idea that machines / systems should access data and learn and explore from this data themselves. It is the recognition of patterns from large data sets.

Examples of Machine Learning:
  1. Google uses Machine Learning to identify pictures.
  2. Voice assistants like Siri or Google assistant are being trained by Machine Learning to recognise when someone is talking and what to respond to that.
  3. Spotify uses Machine Learning to make personalized playlists by using your preferences and the type of songs you listen to.
What is Machine Learning exactly and simple said?
Machine learning is a type of artifical intelligence that occurs when computers become more intelligent without humans explicitly teaching them on how to behave. It enables a self-learing form of data and applies that learing without the need for human intervention. The computer identifies patterns in data and learns from its own mistakes.The 3 main forms of machine learning are: supervised learning, unsupervised learning and reinforcement learning.
Supervised, unsupervised and reinforcement learning
By supervised learning the machine is provided with a lot of information about a certain case and the case outcome is also provided. The outcome is the labeled data and the rest of the information is used as input. An example: the machine is shown a 100 cases when the customers defaulted on a loan and 100 cases where customers didn't default. Things like age and salary are the input, and default/ not- default is the labeled data. With this information the machine can learn the relationships between the outcome default and the input information.
By unsupervised learning the computer must learn without the help from the user. The machine identifies patters in the data that is not obvious for the human eye. It recognises patters and is quite useful to help us make decisions. An example: if we did not know which customers defaulted on loans, but gave the borrower information to the machine, it would be able to see patterns among the borrowers and group them. Unsuperivised learning is used for the detections like uncovering fraudulent transactions or payments.
Reinforcement learning is the closest to the humans when it comes to the way of learning. The algorithm continually learns from its environment by interacting with it. For each action it gets a positive or a negative reward. An example; the bankloans. A reinforcement learning algorithm looks at the information and classifies the custumor as a high-risk customer. When the person defaults, the algorithm gets a positive reward, if the person doesn't default, it gets a negative reward. The rewards help the agent understand the environment and the problem better, and learned for future cases.
Machine learning versus traditional programming
Traditional programming takes out the data, uses the rules as input and applies the rules to the data to get the answers. Whereas in Machine Learning (ML) the data and the answers are used for input and the new learned rules come as output. It lets the computer learn new rules in a complex space, a space harder to comprehend by humans. For example, a traditional computer program for activity recognition for someone's speed (data) and definiton of activity (rules). The problem: different people walk with variating speeds depending upon, for example, age. With ML lots of example activities with their labels will be given and it then learns the rules to predict future activity.
Traditional programming is manually done and if there occurs something like an error, humans would have to fix it. Machine learning on the other hand learns from its own mistakes and can fix those errors without the need for humans to help.
